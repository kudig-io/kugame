## 快速开始

### 创建角色

首次启动游戏时，你需要：

1. **输入侠名**: 选择一个你喜欢的名字
2. **选择门派**: 四个门派各有特色

| 门派 | 特色 | 推荐玩家 |
|-----|------|---------|
| 青云宗 | 经验+10%，防守稳健 | 新手玩家 |
| 玄天宗 | 平衡发展，策略性强 | 喜欢思考的玩家 |
| 炼狱门 | 经验+20%，攻击凶猛 | 喜欢挑战的玩家 |
| 逍遥派 | 经验+15%，自由灵活 | 喜欢探索的玩家 |

### 主菜单

游戏主菜单提供以下功能：

- 📖 **开始故事**: 学习新的 Kubernetes 命令
- ⚔️ **修炼场**: 练习已学命令
- 🏆 **挑战关卡**: 完成挑战获得奖励
- 📝 **知识问答**: 测试知识掌握程度
- 📊 **修炼进度**: 查看当前进度
- 📚 **命令手册**: 查看所有命令
- 🎒 **装备管理**: 查看和强化装备
- 🏪 **仙缘商店**: 购买和出售装备
- 🏰 **副本挑战**: 每日副本和挑战之塔

## 基础系统

### 修炼境界

从凡人到金仙，共 10 个境界：

| 境界 | 等级 | 解锁功能 |
|-----|------|---------|
| 凡人 | 1-10 | 基础功能 |
| 练气期 | 11-20 | 技能系统 |
| 筑基期 | 21-30 | 天赋树 |
| 金丹期 | 31-40 | 装备强化 |
| 元婴期 | 41-50 | 高级副本 |
| 化神期 | 51-60 | 精英副本 |
| 大乘期 | 61-70 | 传说装备 |
| 渡劫期 | 71-80 | 极限挑战 |
| 散仙 | 81-90 | 全副本解锁 |
| 金仙 | 91-100 | 塔顶挑战 |

### 经验获取

- **学习命令**: 每学一个新命令 +50 经验
- **完成挑战**: +100 经验
- **通关章节**: +500 经验
- **副本挑战**: 大量经验
- **挑战之塔**: 层数越高经验越多

**经验加成来源**:
- 门派加成（10%-20%）
- 装备加成（最高 60%）
- 技能加成（青云宗道法自然 +30%）
- 天赋加成（最高 40%）

## 战斗系统

### 战斗流程

1. **遭遇怪物**: 在故事或副本中触发战斗
2. **回答问题**: 正确回答 Kubernetes 命令题
3. **造成伤害**: 答对造成全额伤害，答对一半
4. **怪物反击**: 怪物会反击造成伤害
5. **战斗结束**: 一方生命值归零

### 战斗属性

- **攻击力**: 决定造成伤害
- **防御力**: 减少受到的伤害
- **生命值**: 生命归零则战斗失败

**属性计算公式**:
```
实际攻击 = 基础攻击 + 武器攻击 + 饰品攻击
实际防御 = 基础防御 + 护甲防御 + 饰品防御
实际生命 = 基础生命 + 护甲生命 + 饰品生命
```

### 连击系统

连续答对可获得经验加成：

| 连击数 | 经验加成 |
|-------|---------|
| 1-2 | 10% |
| 3-4 | 20% |
| 5-9 | 30% |
| 10+ | 50%（最高） |

## 装备系统

### 装备类型

| 类型 | 位置 | 主属性 |
|-----|------|-------|
| 武器 | 右手 | 攻击 |
| 护甲 | 身体 | 防御、生命 |
| 饰品 | 配饰 | 综合属性 |

### 装备品质

品质从低到高：

1. **普通**（白色）: 基础属性
2. **精良**（绿色）: 属性 ×1.3
3. **稀有**（蓝色）: 属性 ×1.6
4. **史诗**（紫色）: 属性 ×2.0
5. **传说**（橙色）: 属性 ×2.5

### 获取装备

**途径一：战斗掉落**
- 击败怪物有概率掉落
- 层数/难度越高，掉落品质越好

**途径二：商店购买**
- 使用经验值购买
- 每日自动刷新
- 可手动刷新（消耗100经验）

**途径三：副本奖励**
- 装备副本必掉高品质装备
- 挑战之塔高层奖励

### 装备强化

消耗经验值强化装备：

1. 进入 **装备管理** → **强化装备**
2. 选择要强化的装备
3. 确认强化费用
4. 强化成功，属性提升

**强化规则**:
- 每级提升 10% 属性
- 品质越高，可强化等级越高
- 最高可强化至 +9

### 装备出售

不需要的装备可以出售换取经验：

1. 进入 **仙缘商店** → **出售物品**
2. 选择要出售的装备
3. 确认出售价格
4. 获得经验值

## 技能与天赋

### 门派技能

每个门派有 3 个独特技能：

**青云宗**:
- 稳如泰山（被动）: 减伤 20%
- 道法自然（主动）: 3场战斗经验 +30%
- 以柔克刚（主动）: 反弹下次伤害

**炼狱门**:
- 狂暴（主动）: 本回合攻击翻倍
- 嗜血（被动）: 吸血 20%
- 不屈（被动）: 30% 概率复活

**玄天宗**:
- 变化莫测（主动）: 50% 闪避 2回合
- 天机洞察（主动）: 显示正确答案
- 乾坤挪移（主动）: 转移伤害给怪物

**逍遥派**:
- 逍遥游（被动）: 逃跑 100% 成功
- 因材施教（被动）: 学习命令经验 +50%
- 随性而为（主动）: 随机强力效果

**使用技巧**:
- 被动技能自动生效
- 主动技能有冷却时间
- 合理时机使用主动技能

### 天赋系统

升级获得天赋点，投入天赋树强化角色。

**天赋分支**:
- 攻击: 提升输出能力
- 防御: 提升生存能力
- 辅助: 提升效率收益

**解锁规则**:
- Tier 1: 直接投入（最多5点）
- Tier 2: 需要 Tier 1 投入3点（最多3点）
- Tier 3: 需要 Tier 1+2 投入6点（最多1点）

**加点建议**:
- 新手：优先防御分支，提高容错率
- 进攻：攻击分支，快速击败敌人
- 效率：辅助分支，更快升级

## 副本与挑战塔

### 每日副本

每天可挑战的 3 种副本：

**修炼秘境**:
- 消耗: 10 体力
- 奖励: 1000 经验
- 推荐: 新手每日必做

**藏宝洞窟**:
- 消耗: 15 体力
- 奖励: 500 经验 + 稀有装备
- 推荐: 需要装备时挑战

**绝境试炼**:
- 消耗: 20 体力
- 奖励: 2000 经验 + 史诗装备
- 推荐: 实力较强时挑战

**体力恢复**:
- 每 6 分钟恢复 1 点
- 最大 100 点

### 挑战之塔

100 层无尽挑战，难度递增。

**挑战规则**:
- 从第 1 层开始
- 通关后可挑战下一层
- 每层只有一个怪物
- 每 10 层是 BOSS

**奖励递增**:
- 层数越高，经验越多
- 装备品质越好
- BOSS 层有额外奖励

**排名系统**:
- 根据最高通关层数排名
- 达到特定层数获得称号

**挑战建议**:
- 打不过时先提升装备
- 合理使用技能
- 注意恢复生命值

## 进阶技巧

### 快速升级

1. **优先完成故事章节**: 每个章节 500 经验
2. **保持连击**: 高连击有经验加成
3. **使用经验装备**: 优先选择有经验加成的装备
4. **挑战副本**: 每日副本经验丰厚
5. **青云宗/逍遥派**: 这两个门派有经验加成

### 装备搭配

**输出型**:
- 武器: 高攻击 + 经验加成
- 护甲: 适当防御
- 饰品: 攻击 + 连击加成

**防御型**:
- 武器: 适中攻击
- 护甲: 高防御 + 高生命
- 饰品: 防御 + 生命

**平衡型**:
- 所有装备属性均衡
- 饰品选择综合属性

### 战斗技巧

1. **观察敌人**: 注意敌人的攻击和防御
2. **合理使用技能**: 关键时刻使用主动技能
3. **保持连击**: 尽量连续答对
4. **及时逃跑**: 打不过时及时逃跑保存实力
5. **恢复生命**: 战斗间隙恢复生命值

### 资源管理

**经验值使用优先级**:
1. 升级（自动）
2. 购买装备（提升战力）
3. 强化装备（长期收益）
4. 刷新商店（寻找好装备）

**装备管理**:
- 及时出售不需要的装备
- 保留高品质装备
- 优先强化主武器

### 门派选择建议

**新手推荐**: 青云宗
- 经验加成帮助快速升级
- 减伤被动提高容错率

**挑战推荐**: 炼狱门
- 最高经验加成
- 吸血和复活提高生存

**策略推荐**: 玄天宗
- 闪避和提示降低难度
- 适合喜欢思考的玩家

**探索推荐**: 逍遥派
- 逃跑和随机性增加乐趣
- 装备掉落加成

## 常见问题

**Q: 游戏进度会保存吗？**
A: 会自动保存，也可以手动在菜单中选择保存。

**Q: 可以切换门派吗？**
A: 不能切换，但可以多存档体验不同门派。

**Q: 装备强化失败会怎样？**
A: 装备强化不会失败，只是消耗经验值。

**Q: 挑战塔打不过怎么办？**
A: 先提升等级和装备，或者挑战低层积累资源。

**Q: 体力用完了怎么办？**
A: 等待自然恢复，或者先进行不消耗体力的活动（如故事模式）。
