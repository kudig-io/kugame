# 宠物系统文档

## 概述

宠物系统为玩家提供陪伴和战斗辅助，玩家可以通过培养、进化来提升宠物的能力。

## 宠物类型

### 按功能分类

| 类型 | 特点 | 代表宠物 |
|------|------|---------|
| 攻击型 | 高攻击，输出能力强 | 烈焰虎、雷霆雕 |
| 防御型 | 高防御，能承受伤害 | 玄铁龟 |
| 辅助型 | 治疗、增益技能 | 灵狐 |
| 平衡型 | 属性均衡，全面发展 | 碧玉龙 |

### 按稀有度分类

| 稀有度 | 颜色 | 属性加成 | 获取难度 |
|--------|------|---------|---------|
| 普通 | 白色 | 1.0倍 | 容易 |
| 稀有 | 绿色 | 1.2倍 | 较易 |
| 精良 | 蓝色 | 1.5倍 | 中等 |
| 史诗 | 紫色 | 2.0倍 | 较难 |
| 传说 | 橙色 | 3.0倍 | 困难 |
| 神话 | 红色 | 5.0倍 | 极难 |

## 宠物列表

### 灵狐（稀有/辅助型）
- **外观**：一只通体雪白的狐狸，眼神灵动
- **初始属性**：攻8 防6 血80
- **技能**：
  - 治愈之吻：恢复主人生命值
  - 灵动：提升主人速度
- **成长方向**：辅助、治疗

### 烈焰虎（精良/攻击型）
- **外观**：一只浑身燃烧着火焰的猛虎
- **初始属性**：攻15 防8 血120
- **技能**：
  - 火焰爪：造成火焰伤害
  - 狂暴：提升攻击力
- **成长方向**：高攻击输出

### 玄铁龟（精良/防御型）
- **外观**：一只背甲如玄铁般坚硬的巨龟
- **初始属性**：攻5 防15 血150
- **技能**：
  - 铁壁：提升防御力
  - 嘲讽：吸引敌人攻击
- **成长方向**：坦克、肉盾

### 雷霆雕（史诗/攻击型）
- **外观**：一只羽翼带电的金色大雕
- **初始属性**：攻20 防10 血100
- **技能**：
  - 雷霆一击：造成雷电伤害
  - 疾风：提升速度
- **成长方向**：爆发输出

### 碧玉龙（传说/平衡型）
- **外观**：一条通体碧玉的小龙，散发着威严的气息
- **初始属性**：攻18 防18 血180
- **技能**：
  - 龙息：造成巨大伤害
  - 龙威：全面提升属性
  - 龙血：恢复大量生命
- **成长方向**：全面发展

## 培养系统

### 升级
- 宠物通过获得经验升级
- 升级提升基础属性
- 升级所需经验随等级增加

### 进化
- **进化条件**：达到指定等级（20/40/60级）
- **进化效果**：
  - 大幅提升属性（1.5倍/级）
  - 提升成长率
  - 外观变化（幼体→成长→成熟→完全体）

### 喂食
- 增加忠诚度（影响战斗表现）
- 增加心情
- 恢复生命值
- 食物品质1-5，效果递增

### 玩耍
- 提升心情
- 少量增加忠诚度
- 心情影响训练效果

### 训练
- 获得经验值
- 针对性提升属性
  - 力量训练：+攻击
  - 耐力训练：+防御
  - 体质训练：+生命
- 消耗心情

## 战斗系统

### 宠物技能
- 每只宠物拥有1-3个技能
- 技能有冷却回合
- 技能类型：攻击、治疗、增益、减益

### 出战机制
- 只能同时出战1只宠物
- 出战宠物跟随玩家战斗
- 宠物属性按稀有度加成后计入总战力

### 技能使用
```python
# 使用第一个技能
result = pet.use_skill(0)
if result["success"]:
    print(f"使用了 {result['skill_name']}!")
```

## API接口

### 创建宠物
```python
from kugame.pet_system import create_pet, PetManager

# 从模板创建
pet = create_pet("spirit_fox")

# 生成随机宠物
random_pet = generate_random_pet(
    min_rarity=PetRarity.稀有,
    max_rarity=PetRarity.传说
)
```

### 宠物管理器
```python
manager = PetManager()

# 添加宠物
manager.add_pet(pet)

# 设置出战宠物
manager.set_active_pet(pet.id)

# 获取出战宠物
active_pet = manager.get_active_pet()

# 获取宠物摘要
summary = manager.get_summary()
```

### 培养操作
```python
# 获得经验
leveled_up = pet.gain_exp(100)

# 喂食
result = pet.feed(food_quality=3)

# 玩耍
result = pet.play()

# 训练
result = pet.train("attack")

# 进化（自动在升级时检查）
pet.evolve()
```

## 与Player类集成

```python
class Player:
    def __init__(self):
        # ...其他初始化...
        self.pet_manager = PetManager()
    
    def get_active_pet_bonus(self) -> Dict[str, int]:
        """获取出战宠物属性加成"""
        pet = self.pet_manager.get_active_pet()
        if not pet:
            return {"attack": 0, "defense": 0, "health": 0}
        
        return {
            "attack": pet.total_attack // 10,  # 10%加成
            "defense": pet.total_defense // 10,
            "health": pet.total_health // 10,
        }
```

## 获取途径

1. **任务奖励**：完成特定任务获得
2. **商店购买**：使用金币或钻石购买
3. **活动获取**：限时活动赠送
4. **野外捕捉**：随机事件中获得
5. **进化获得**：低级宠物进化

## 后续扩展

1. **宠物装备**：为宠物装备专属装备
2. **宠物融合**：两只宠物融合产生新宠物
3. **宠物图鉴**：收集所有宠物获得奖励
4. **宠物家园**：宠物休息和互动场所
5. **宠物竞技**：宠物之间的对战
