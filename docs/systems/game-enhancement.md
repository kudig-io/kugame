# 游戏性加强文档

## 概述

本次游戏性加强为KuGame添加了多个深度系统，大幅提升游戏的可玩性和耐玩性。

## 新增系统

### 1. 任务系统

#### 每日任务
- 每天自动生成3个随机任务
- 任务类型包括：学习命令、完成战斗、答题、收集装备等
- 完成后获得经验、体力等奖励
- 每日0点自动重置

#### 每周任务
- 每周生成2个高难度任务
- 奖励比每日任务更丰厚
- 每周一0点自动重置

#### 主线任务
- 7个章节任务，推进剧情发展
- 有等级和前置任务要求
- 完成获得大量经验和专属称号

#### 支线任务
- 5个特色任务
- 包含完美连击、副本探索、传说收集等挑战
- 完成获得稀有奖励

**文件位置**:
- `kugame/quest_system.py` - 核心任务系统
- `kugame/main_quests.py` - 主线和支线任务定义

### 2. 装备套装系统

| 套装名称 | 件数 | 品质 | 适用门派 |
|---------|------|------|---------|
| 新手套装 | 3件 | 普通 | 通用 |
| 青云套装 | 3件 | 精良 | 青云宗 |
| 炼狱套装 | 3件 | 精良 | 炼狱门 |
| 玄天套装 | 3件 | 精良 | 玄天宗 |
| 逍遥套装 | 3件 | 精良 | 逍遥派 |
| 龙鳞套装 | 4件 | 稀有 | 通用 |
| 凤羽套装 | 4件 | 史诗 | 通用 |
| 混元套装 | 5件 | 传说 | 通用 |

#### 套装效果
- 2件套：基础属性加成
- 3件套：特殊属性加成（经验、暴击、闪避等）
- 4/5件套：终极属性加成

**文件位置**: `kugame/equipment_sets.py`

### 3. 宝石系统

#### 宝石类型
- 红宝石：攻击加成
- 蓝宝石：防御加成
- 绿宝石：生命加成
- 黄宝石：经验加成
- 紫宝石：连击加成
- 橙宝石：暴击率加成

#### 宝石品质
- 普通（白）：1.0倍效果
- 精良（绿）：1.5倍效果
- 稀有（蓝）：2.0倍效果
- 史诗（紫）：3.0倍效果
- 传说（橙）：5.0倍效果

#### 宝石合成
- 同类型、同品质、同等级的两颗宝石可合成
- 合成后等级+1
- 达到品质上限后提升品质

**文件位置**: `kugame/gem_system.py`

### 4. 排行榜系统

#### 排名维度
- 总经验值排名
- 等级排名
- 成就数量排名
- 挑战塔层数排名
- 战斗胜场排名
- 战斗胜率排名

#### 功能特性
- 实时更新排名数据
- 查看前N名玩家
- 查看个人排名详情
- 排行榜统计数据

**文件位置**: `kugame/leaderboard.py`

### 5. 事件链系统

#### 事件链
1. **炼丹奇遇**（稀有）
   - 触发：山中偶遇炼丹师
   - 分支：小心看守/加入材料/婉言谢绝
   - 结果：获得经验丹/神秘变异/无事发生

2. **古墓探险**（史诗）
   - 触发：发现隐秘古墓
   - 分支：勇敢进入/做好准备
   - 流程：走廊探索 → 分岔路口 → 宝藏/战斗/灵泉

3. **仙人指点**（传说）
   - 触发：云游仙人出现
   - 分支：恭敬行礼/求教大道
   - 结果：获得仙石/仙人离去

#### 独立事件
- 意外之财：发现宝箱
- 神秘商人：购买稀有物品
- 落难弟子：施以援手

**文件位置**: `kugame/event_chains.py`

## 系统集成

### Player类集成
所有新系统都已集成到Player类：

```python
# 任务系统
player.refresh_daily_quests()
player.update_quest_progress('learn_command', 1)
player.get_available_main_quests()

# 装备套装（自动计算）
player.total_attack  # 包含套装加成

# 宝石系统
player.gem_inventory  # 宝石背包

# 排行榜
player.update_leaderboard_ranking()

# 事件链
player.event_chain_manager.trigger_random_event()
```

### GameEngine集成
在游戏关键节点自动触发系统功能：
- 战斗胜利 → 更新任务进度
- 获得装备 → 检查套装效果
- 每日登录 → 刷新每日任务
- 学习命令 → 更新任务进度
